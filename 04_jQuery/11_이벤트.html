이벤트 핸들러(이벤트 발생 시 실행할 function) 연결 방식

* 방법1. 이벤트 메소드 통한 연결

        $("선택자").이벤트메소드(function(){
            선택자에 해당 이벤트 메소드 발생했을 때 실행할 내용;
        })

클릭해보세요



* 방법2. on 메소드를 이용한 방법

    $("선택자").on("이벤트명", function(){
        해당 선택자에 이벤트 발생 시 실행할 내용;
    });

마우스 클릭 및 올려보세요



* 방법3 

    $("상위요소선택자").on("이벤트명","하위요소선택자(이벤트를 걸고자하는 요소)", function(){
        // 이걸 꼭 써야하는 경우가 생김
        선택된 상위요소 안에 존재하는 하위요소에 해당 이벤트 발생 시 실행할 내용
    })

h4를 클릭해보세요

h5를 클릭해보세요
div 바깥쪽 h5

* 키보드 관련 이벤트

- keydown | keypress : 키가 눌러질 때 발생하는 이벤트
> keydown : 키보드의 모든 키가 눌러질 때 발생됨
> keypress : 키보드의 펑션키(F1,F2..), 기능키(컨쉬알), 한글 제외한 나머지 키가 눌러질 때 발생됨
- keyup : 키가 떼어질 때 발생하는 이벤트
