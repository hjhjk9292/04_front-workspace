color : 텍스트의 색상을 지정

    선택자{
        color : 색상명 | 16진수 | rgb(x,x,x) rgba(x,x,x,투명도) | hsl(x,x,x) 채도명도, hsla(x,x,x,투명도)
    }

text-decoration : 텍스트에 줄을 긋거나 줄을 없앨 때 사용

    선택자{
        text-decoration : none | underline | overline | line-through;
    }

text-transform : 영문 텍스트의 대소문자 변환 시 사용

     선택자 {
        text-transfrom : uppercase | lowercase | capitalize
     }
  1. uppercase : 모든 영문자 다 대문자로
  2. LowerCase : 모든 영문자 다 소문자로
  3. capitalize: 영문자의 firsttext만 대문자로

text-shadow : 텍스트에 그림자 효과를 줄 때 사용

    선택자 {
        text-shadow : 가로거리(x) 세로거리(y) [번짐정도] [색상]
    }
